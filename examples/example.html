Teken een 1

We gaan de pen gebruiken om een 1 te tekenen. Ik toon hier alvast hoe het moet, maar probeer het ook eens zelf.

when gf clicked set pen shade to {0} point in direction {90} pen down move {8} steps turn cw {90} degrees move {25} steps turn cw {90} degrees move {8} steps turn cw {180} degrees move {16} steps turn ccw {90} degrees move {8} steps pen up

Teken een 0

Kan je nu ook een 0 tekenen? Merk op dat je 2 keer dezelfde blokjes achter elkaar zet, je kan dit vermijden door een blokje te gebruiken.

when gf clicked set pen shade to {0} point in direction {90} pen down repeat 2 move {12} steps turn cw {90} degrees move {22} steps turn cw {90} degrees end pen up

Van een vierkant naar een cirkel

Teken nu een vierkant. Ik heb dit zo gedaan:

when gf clicked set pen shade to {0} point in direction {90} pen down repeat 4 move {10} steps turn cw {({360}/{4})} degrees end pen up

Wat gebeurt er als je vervangt door ? Wat als we hier nu een groot getal van maken, bijvoorbeeld ?
Nu kunnen we een cirkel tekenen!

when gf clicked set pen shade to {0} pen down repeat ({30}/{2}) move {5} steps turn cw {({360}/{30})} degrees end pen up
