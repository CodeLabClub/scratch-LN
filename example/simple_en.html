Draw figures

In the example below we use the variable

(counter)

to draw a polygon with different number of sides.

Draw polygon

when greenflag clicked go to x: {0} y: {0} pen down repeat (counter) move {100} steps turn cw {({360}/{(counter)})} degrees end pen up
